# bootcamp-aws

## 1º Desafio
Elaborar e desenhar a arquitetura de um sistema na AWS utilizando o draw.io, praticando os conceitos que foram abordados até o momento no bootcamp.

O sistema apresentado representa o fluxo de uma API REST que está hospedado em uma EC2 na AWS Cloud, a EC2 se comunica com um banco de dados e com uma EBS que terá Snapshots EBS regurlamente guardados em um S3 bucket.
